What is the differnece between Web 1, Web 2 and Web 3?

Well, To answer that question lets explain each one of the Web versions:

Web 1: One of the primary version of the web as we know it. In this version the web users had a limited intercation between them and the website itself. Also websites barley didn't have intercation with one another. Users could not post or comment feedbacks. Web 1 was a good way to let business let people know that they were out there and present information about their business.

Web 2: In this version of the web users could interact with the website itself - they could comment, post and leave feedback. Web 2 encourged participation, collabration and information sharing from the web user. In this web version the users got introduced to website such as Facebook, Youtube and Wikipedia. Using Web 2 people could comminucate with each other and share experinces from their daily life. This version is the web as most people know it. People could now also search what they wanted to view using search engines. Another thing that charchertizes Web 2 is the web broswsers. Companies start to create web broswesers such as Internet Explorer, Google Chrome and many more.
In oppose to Web 1, which was considred to be an only "read" content, Web 2 is a "read-write" content.

Web 3: So what else can we ask for that wans't implemented in Web 2? Web 3 is all about interperting information like human beings can. If we refred to Web 1 as "read" content and Web 2 as "read-write" content we can refer to web 3 as a "read-write-execute" content. We want the machine to perform on it's own - the computer will act as if it is a personal assistant. The web 3 is the web version which we are using now-days.
